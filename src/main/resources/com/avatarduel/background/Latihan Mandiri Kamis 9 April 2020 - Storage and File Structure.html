Question 3

Complete
Mark 20.00 out of 20.00
Flag question

Question text

Tentukan organisasi file yang paling sesuai dengan pernyataan tentang penempatan record pada file yang diberikan.

Record-record di dalam file ditempatkan berdasarkan hasil perhitungan terhadap sejumlah atribut record yang akan menentukan pada block data mana record akan ditempatkan.

Record-record di dalam file ditempatkan berdasarkan keterurutan atribut kunci pencarian (search-key) dari masing-masing record.

Record-record di dalam file ditempatkan berdasarkan urutan kedatangan.

Record-record dari relasi yang berbeda dapat disimpan pada file yang sama.

Feedback

The correct answer is: Record-record di dalam file ditempatkan berdasarkan hasil perhitungan terhadap sejumlah atribut record yang akan menentukan pada block data mana record akan ditempatkan. → Organisasi File Hashing, Record-record di dalam file ditempatkan berdasarkan keterurutan atribut kunci pencarian (search-key) dari masing-masing record. → Organisasi File Sequential, Record-record di dalam file ditempatkan berdasarkan urutan kedatangan. → Organisasi File Heap, Record-record dari relasi yang berbeda dapat disimpan pada file yang sama. → Organisasi File Multitable Clustering
